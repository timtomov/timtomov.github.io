# timtomov.github.io

Personal academic website for Tim Tomov, built with [al-folio](https://github.com/alshedivat/al-folio) and deployed to GitHub Pages.

## Local preview

```bash
/opt/miniconda3/bin/conda run -n publication bundle install
/opt/miniconda3/bin/conda run -n publication bundle exec jekyll serve
```

The site is then available at the local URL printed by Jekyll, usually `http://127.0.0.1:4000/`.
